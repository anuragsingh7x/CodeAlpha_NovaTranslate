# 🌍 NovaTranslate

An AI-powered multilingual language translation web application built using Python, Flask, HTML, CSS, and JavaScript.

## ✨ Features

- 🌍 Translate text across 19+ languages
- 🔄 Swap source and target languages
- 📋 Copy translated text
- 🔊 Text-to-Speech
- 🌙 Dark Mode
- 📱 Responsive UI
- ⚡ Fast translation using deep-translator

## 🛠️ Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- deep-translator
